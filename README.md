# The Needy Jhansi — Geospatial Directory

An interactive, responsive geospatial web application designed to help residents and tourists discover essential services, educational institutions, monuments, and amenities in Jhansi, India.

### Key Features
* **Interactive Mapping:** Powered by Leaflet.js and OpenStreetMap (using CartoDB Voyager tiles).
* **Client-side Search & Filtering:** Instant keyword search and multi-level category filters (Government vs. Private, Shopping types, etc.).
* **State Persistence:** Uses HTML5 LocalStorage to allow users to add custom locations that persist across page refreshes.
* **Responsive Layout:** Built with Tailwind CSS, optimized for mobile devices and desktops.

### Technology Stack
* Frontend: HTML5, Tailwind CSS, JavaScript (ES6)
* Maps: Leaflet.js API, OpenStreetMap, CartoDB Tiles
* Storage: Browser LocalStorage (No database installation required for local demo)
